# tinker-ocr

An OCR plugin for [TINKER](https://github.com/liriliri/tinker), powered by Tesseract.js.

![Screenshot](https://raw.githubusercontent.com/liriliri/tinker-plugins/master/packages/tinker-ocr/screenshot.png)

## Features

- **Image input** via file upload, drag & drop, paste, or screen capture
- **Multi-language** support (Chinese, English, Chinese + English)
- **Strip newlines** option for cleaner output
- **Copy or save** recognized text

## Installation

Download and install TINKER from `https://tinker.liriliri.io/`, then run `npm i -g tinker-ocr`.

## Usage

1. Upload an image by clicking, dragging & dropping, pasting (Ctrl+V), or using the capture screen button
2. Select the recognition language
3. Click **Recognize** to extract text
4. Copy or save the result
